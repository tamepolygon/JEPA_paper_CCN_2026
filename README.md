# Prototype Structure in Self-Supervised Vision Models

CCN 2026 extended abstract on prototype structure in self-supervised
vision representations.

## Summary

We probe whether self-supervised vision models (I-JEPA) encode graded
category structure, testing if representations reflect human typicality
judgments. Across 1,854 THINGS concepts, we find a typicality
dissociation (Δr = +0.541), indicating that prototype-like structure
emerges without category labels.
